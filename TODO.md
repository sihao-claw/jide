## 📝 开发注意事项

1. **AI 服务**: 需要配置实际的 API Key（替换 `lib/services/ai_service.dart` 中的占位符）

2. **通知权限**: 
   - Android 13+ 需要动态请求通知权限
   - iOS 需要在 Info.plist 中配置

3. **打包要求**:
   - Windows: 使用 Inno Setup 或类似工具打包成单 exe
   - Android: 直接生成 APK
   - iOS: 暂不要求

4. **自测流程**:
   - 每次开发后构建 Linux 版本
   - 使用 computer-use 技能进行自动化测试
   - 确认功能正常后再提交代码
